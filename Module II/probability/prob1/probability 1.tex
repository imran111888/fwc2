\section{problem }

compute $P(A | B)$,IF $P(B)=0.5$ and $P(AB)=0.32$ ?

\section{solution }

By using property of conditional probability we have,
\begin{align*}
P(A | B)=\frac{P(AB)}{P(B)}=\frac{0.32}{0.5}=0.64
\end{align*}
