\section{problem }

Let A and B be independent events with $P(A)=0.3$ and $P(B)=0.4$ find 
\begin{enumerate}

\item$P(AB)$
\item$P(A+B)$ 
\item$P(A | B)$   
\item$P(B | A)$
       \end{enumerate}
\section{solution }  


Since A and B are independent events, we have 

\begin{enumerate}

\item $\begin{aligned}
 P(AB)&=P(A) P(B)&\\
P(AB)&=0.3 \times 0.4&\\
P(AB)&=0.12&
\end{aligned}$


\item $\begin{aligned}
P(A+B)&=P(A)+P(B)-P(AB)&\\
P(A+B)&=0.3+0.4-0.12=0.58&
\end{aligned}$


\item $\begin{aligned}
P(A | B)=\frac{0.12}{0.40}=0.3
\end{aligned}$
\item $\begin{aligned}
P(B | A)&=\frac{P(B+A)}{P(A)}&=\frac{0.12}{0.30}&=0.4&
\end{aligned}$
\end{enumerate}
