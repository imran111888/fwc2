\section{problem }

Compute $\pr{A|B}$, if $\pr{B}=0.5$ and $\pr{AB}=0.32$ ?

\section{solution }

By using property of conditional probability we have,
\begin{align}
\pr{A|B}=\frac{\pr{AB}}{\Pr{B}}=\frac{0.32}{0.5}=0.64
\end{align}
